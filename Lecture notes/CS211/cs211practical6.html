CS21120 Practical 6: Binary Trees

Introduction

This exercise will help you understand the implementation of trees. To gain maximum benefit, do not use the code produced in lectures. By all means look at the code provided but use it to generate psuedo-code and re-implement from that.

This problem is to use a Binary Search Tree to make a concordance of words giving details on where they occur.

This will give you some experience of using Binary Search Trees, of recursion and of the StreamTokeniser class.

 

The idea of this program is that you take a file and from it produce a concordance of the occurrences of indexed words in the file.

 

Typical input (from a text file which includes linebreaks):

 

Forced to do his homework in the dead of night and forbidden to refer to

his \index{magic} skills or his life at \index{Hogwarts} School,  \index{Harry} is forced to

endure the summer holidays with the dreaded \index{Dursleys}.

Running away from his dreaded relatives, \index{Harry} expects to be

expelled from \index{Hogwarts} for his blatant flaunting of the rule not to use

\index{magic} outside term time. \index{Magic} is strictly forbidden outside school.

 

Resulting output:

 

        Words Indexed:

        dursleys      line: 3

harry         line: 2,4

hogwarts     line: 2,5

magic        line: 2,6             Note: although magic is indexed twice on line 6, it only gets shown once
