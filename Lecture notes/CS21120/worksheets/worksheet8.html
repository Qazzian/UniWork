CS21120 Worksheet 8: Creating a state diagram for the Coin Handling and Dispensing Unit class

Introduction

This exercise will help you understand the notation used in state diagrams, the process used during the Design workflow to construct them from a design sequence diagram.

This will be achieved by developing a state diagram for the boundary class CoinHandling&DispensingUnit (CDHU) based on information in the Obtain Confectionery Item sequence diagram.

Steps

  1. Follow the process steps laid presented on the course slide entitled "Process of Building State Diagrams". For this exercise choose the CDHU class. Draw a simplified state diagram that omits behaviour concerned with alternative courses of action (the inclusion and/or exclusion use cases of the OCI use case).
  2. Draw the diagram by hand first and then add a state diagram to the Logical view package. Name the diagram Coin Handling and Dispensing Unit.
