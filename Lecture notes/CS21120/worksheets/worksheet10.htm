CS21120 worksheet 10: Graphs

 

Introduction

 

This exercise will help you gain a better understanding of how the graph ADT operates by getting you to implement one.

 

The lectures have provided you with four interfaces: the graph, the edge, the vertex and the dictionary.  These can be found at http://users.aber.ac.uk/jit93/teaching if you do not have them.

 

These interfaces provided for a graph implemented using an adjacency list, i.e. an dynamic data structure.  The tasks are :

 

  1) Implement these interfaces.  You will find that you will need to add extra private/protected methods to many of these as you develop  the class.

 

  2 Provide a suitable test harness for your code.  The test harness can either take input from a text file, for an example see below, or from the keyboard.  My recommendation is that you use a text file, but that is your choice.  Each line is an edge in the graph, for example A is a vertex in the graph.

 

  3)  Some of the methods in the interfaces should be written with exception.  Identify these and add exceptions that are suitable.  You may need to define your own.

 

  4) Extend your code so that more information can be placed into a vertex object - rather than just the ID.

 

Example Text File.

 

A,B,10

B,C,9

D,C,10

B,D,15

B,E,9

C,E,10

D,A,20
