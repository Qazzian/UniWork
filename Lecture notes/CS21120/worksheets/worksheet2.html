CS21120 Worksheet 2: Hashing

Hashing

 

We have spent a fair amount of time looking at hashing and in particular looking at how to maximize the efficiency of the data structure by considering the type of hashing function, the size of table, the collision resolution techniques (linear versus quadratic probing) and open versus closed addressing.

 

All the code that you need for the various implementations is given in the module web pages but can you reproduce it? If not, will you be able to do so under examination conditions?

 

Once you are familiar with the implementation, I would like you to do some benchmarking. That is, using a simple simulator to generate data, carry out a series of benchmarks to measure the performance of your algorithm by altering all the variable aspects described above. This information should prove to be a valuable asset in assisting your understanding of the algorithms employed. Make sure that you discuss your findings with the demonstrators. Are there any surprises?

 

Still not motivated enough to do the worksheet? Why not try producing a graphical simulation of the hash table showing the hashing function being applied to a key, possible collisions occurring and the subsequent collision resolution (using linear and quadratic probing). If you are still not motivated, then what can I say?

 

Note on worksheets for CS21120

 

You do not need to do this worksheet if you think you understand all the material and can reproduce what is required in an examination. On the other hand if you have difficulty with the above, concentrate on the Hashing code itself and seek plenty of advice from the demonstrators and advisors.
