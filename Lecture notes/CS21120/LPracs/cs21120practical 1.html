Write a version of the Queue class that implements Queue as a Vector and also has an iterator.

Some of this is done in the notes. Your task is to complete it.

Here might be a beginning (from the Vector notes)

import java.util.*; //gets Vector and Iterator

public class Queue implements Iterator

{

private Vector theQueue = new Vector();

/* And now to the operations */

public void addTo (Object item) {

theQueue.addElement(item);

}

public Object removeFrom() {

Object temp = theQueue.firstElement();

theQueue.removeElementAt(0);

return temp;

}

public int lengthOf() {

return theQueue.size();

}

}

//Could be used like this!

public static void main (String args[]) {

Queue q = new Queue();

q.addTo("mary");

q.addTo("bill");

q.addTo("sue");

String first=(String)q.removeFrom();

System.out.println(first);

}

}
