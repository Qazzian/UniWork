Definition 1   Consider two sets and , then a function is a mapping from one set to the other which associates with each element of one and only one element of .

The set is called the domain of the function. The set is called the co-domain. We say that each element of the first set is mapped onto its image in the second set. The set of all images is called the range and it will be a subset of the co-domain. We usually denote functions by the symbols , , , etc.

Note that a function can map more than one element of the domain onto the same element of the range, e.g., Ann and Carol are both mapped onto the element 4. Such functions are said to be many-to-one. Functions which are such that each element of the domain is mapped onto a different element of the range are said to be one-to-one Relationships which are one-to-many can occur, but from our definition of a function, they are not functions.
