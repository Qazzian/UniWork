High-level JMF architecture

Managers

A key component of the JMF is a set of managers which simplify the use of the API and provide co-ordination of its various parts.

  • Manager

    The manager called Manager handles the construction of players, processors, DataSources and Datasinks.

  • PackageManager

    The PackageManager maintains a registry of packages that contain JMF related classes.

  • CaptureDeviceManager

    The CaptureDeviceManager maintains the registry of available capture devices.

  • PlugInManager

    The PlugInManager maintains the registry of available plug-ins.
