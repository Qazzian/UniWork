Introduce a Policy File associated with the server

Produce a file, called ``accept.policy'' containing the single line:

grant {permission java.net.SocketPermission "<IP address>", "accept";};
where the <IP address> is obtained from the error message obtained in step 2. Do not include the port number (the number after the colon). Why not?

Now try to run the server by giving the command:

java -Djava.security.manager -Djava.security.policy=accept.policy PoemServer
on the server machine (and the same command as before on the client machine).
