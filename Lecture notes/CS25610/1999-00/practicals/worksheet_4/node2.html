Your Task

This worksheet asks you to run some Java programs, and then introduce the Java security mechanisms to first prevent the programs from running but then allow them to run with fine-grained access control.

For the purposes of this practical, start both the server and the client on the workstation at which you are sitting. It will be easier to see what is happening if you have one terminal window in which you are running the server and another in which you are running the client. Therefore, in what follows, interpret ``client machine" to mean the window in which you are running the client and the phrase ``server machine" to mean the window in which you are running the server.

Remember you will need to ``kill" one copy of the server before you can start another one. Please make sure both the client and the server are killed before you log off.
