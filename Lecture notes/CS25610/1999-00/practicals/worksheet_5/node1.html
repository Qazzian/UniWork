Introduction

This worksheet is intended to provide some experience of using the Java Media Framework (JMF). As has already been explained in an email, rather than getting you to create applications from scratch, I'm going to ask you to run some of my applications which have been discussed in lectures, observe the facilities that the JMF provides and perhaps fix a few of my bugs!

You will very much need to consult your lecture notes, the listings I have given out, and on-line resources linked from the course materials web site at

http://www.aber.ac.uk/dcswww/Dept/Teaching/Courses/CS25610/

to complete this work.
