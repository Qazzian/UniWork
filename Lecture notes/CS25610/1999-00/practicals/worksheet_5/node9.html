Running Remotely on stonkin

Start a new window on stonkin as described above. If necessary, source the /dcs/dap/jmf.bashrc file in that new window. Change to the directory in which you are working. Now run SimplePlayer again. (You will NOT need to re-compile, 'cuz it's Java and remember ``write once, run anywhere" :-) ).

You will notice that you get no sound. Why not ?? Check the output display in the window in which the software was started. You should be able to locate the problem. You will also find information in the jmf.log file that will have been created.
