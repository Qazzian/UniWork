Your Task

This worksheet uses the same essential scenario as used on worksheet one, namely it asks you to produce first a ``client" program and then later a ``server" program. The problem being solved is again an extension of the PoemServer example presented in lectures.

For this worksheet however, your clients and servers will use Java RMI to communicate with each other.

I provide a sample RMI version of the Poem server which is running on the computer moin.dcs.aber.ac.uk. The rmiregistry is running on moin on port 8000 and the Poem Server's RMI service name is PoemServer.
