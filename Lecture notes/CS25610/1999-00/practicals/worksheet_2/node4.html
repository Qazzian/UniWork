The Client Program

You should construct a client program that can contact my sample RMI server and make appropriate method calls and display the returned results.

The client program should commence by prompting the user for a hostname and RMI port number to contact to reach the PoemServer.

The client program should then prompt the user for requests.

The client program should enable the user to specify a request by some other means that you define. Your client program should analyse the user's request and make appropriate method calls to the RMI Poem Server, displaying the results returned. The user should be allowed to type a sequence of requests and see their respective results via a single run of your client program.
