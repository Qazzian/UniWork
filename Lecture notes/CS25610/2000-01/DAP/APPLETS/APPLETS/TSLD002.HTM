Java Applets

  • An applet is a special type of Java program designed to run inside a web browser or an applet viewer.
  • Applets are one way to create “live” web pages.
  • Applets are constrained in what they are allowed to do. Remember you may be downloading and executing “untrusted” code on your computer.
