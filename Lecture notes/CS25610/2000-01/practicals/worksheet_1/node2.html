Your Task

The overall idea is to create a set of web pages together with a Java Applet that allow a user to ``load" other files and either display them in browser windows and frames or to play them via the audio system.

An example of the overall screen layout of my solution to this worksheet is attached at the end of this document, or available on-line on the course materials web site. Your application does not have to look identical to mine, but it should have the same features and if complete, the same hierarchy of web pages.
