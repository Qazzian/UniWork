2. Use of Postgres

The following description is a simplification of full Postgres facilities, reflecting the local installation decisions.

Users must be registered under Postgres. Each user has as many databases as they require. Each database has a password. (Note that users do not therefore have passwords directly.) New users and new databases must be created by the system administrator. Users cannot create their own databases. Tables in one database cannot be accessed from another database.

Each student registered for CS27310 has been registered. Each has had one database created. Its name is the same as the student's UID. Student's passwords are the same as their UNIX passwords.

The databases have no content.
