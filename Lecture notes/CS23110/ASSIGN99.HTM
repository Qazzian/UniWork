CS23110 Small Real Time Systems.

Assignment 1998-9.

To be completed by 3rd May 1999 and handed in between 2-00 pm and 3-00 pm on that day.

A small supermarket has four checkouts. Each checkout has:

  • a bar code reader to identify the items purchased,
  • a till on which the items and prices are displayed, and which has only a 'TOTAL' button that is pressed when a customer's items have all been processed,
  • a printer on which an itemised bill is printed when the 'TOTAL' button has been pressed,
  • a communications link to a central computer system that holds the price information for each item in the supermarket and which is interrogated for each item processed by a checkout.

The central computer system has a terminal that permits the manager to view the data, to revise prices, and to add and delete items.

Draw a complete set of MASCOT 3 diagrams for this system. Label your diagrams so as to indicate clearly the purpose of each component. Do not add any further explanation of your design.

Jem Rowland, 15 March 1999
