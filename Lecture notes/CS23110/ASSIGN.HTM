A small supermarket has four checkouts.  Each checkout has a bar-code reader and a till.  There is a central database  that holds the bar code numbers, prices and descriptions of all the products in the supermarket.  The manager has a terminal that allows the database to be updated.  As each item is scanned by the checkout operator, a list of that customer’s purchases, with prices, is created.  When the customer’s last item has been scanned, the operator presses a ‘total’ button on the till and the bill is printed out by the till.  Draw a complete set of MASCOT diagrams for this system and label each component in such a way as to indicate clearly its purpose.  Do not give any other explanation of your design.  Avoid the use of SIDAs, using only channels or pools to represent shared data structures.
