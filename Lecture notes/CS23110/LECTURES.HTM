CS23110 Small Real-Time Systems

Lecture Materials

Via the links below you will find material to support the lecture course. It is based on the overhead transparencies that I use. In making it available to you I emphasise that:

  • it is not intended to be a summary of the lectures,
  • reading it is no substitute for attending the lectures, which contain additional material,
  • I may make changes to some of this material as the course progresses.

The lecture material is available under the following (approximate) headings:

  • Introduction to the concepts of real time embedded systems
  • Hardware, software and timing aspects
  • Operating system considerations
  • Introduction to MASCOT 3
  • Extract from the Official Handbook of MASCOT
    • (This is a PDF file of about 1.2MB.)
  • Design using MASCOT 3.
  • Other design methods: HOOD, UML
  • Languages for real-time
  • Development tools
  • Hardware/software tradeoffs in system design
  • Safety aspects, verification and validation, and relevant standards.
