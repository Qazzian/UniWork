Object-Oriented Programming and Design

Brief Description

A major development in software design in the last decade has been the adoption of object-oriented techniques. The object approach first made an impact in the area of programming language design. The ideas have rapidly spread and object-oriented techniques are now of growing importance in areas as diverse as software design methods, databases, graphical user interfaces and distributed systems. In this module an introduction to the key concepts and features of object technology is provided. Students will learn object-oriented analysis and design and will implement object-oriented programs in Java, a modern object-oriented programming language.

Aims

 The aim of this course is to provide an outline of the key features of object-oriented technology. The course has a practical rather than theoretical emphasis: students will learn object-oriented design techniques by building object-oriented programs using a modern object-oriented programming language (Java). As well as lectures, the course will use practicals and web-based learning material.

Objectives

On successful completion of this module, students should:
  • understand the basic concepts of object technology;
  • be familiar with a standard object-orientated modelling notation;
  • have an appreciation of object-oriented analysis and design methods;
  • have a working knowledge of an object-oriented programming language.

Syllabus
Introduction to Object Orientation - 3 Lectures
These lectures introduce the basic concepts of object orientation, such as class, inheritance, method, instance, polymorhism and abstraction. They will also attempt to highlight the difference between an object oriented approach and a procedural approach to software development. Where appropriate comparisons with Ada and the approach it takes will be made.
Introduction to the Java Language - 1 Lecture
This will look at its history, its pedigree and explain the difference between applications and applets.
Basic Java - 2 Lectures, 2 Practicals
These lectures and practicals will introduce the basics of the language such as class definitions, control structures, keywords, interface specifications etc. They will also look at some example applications.
The Java Class library - 1 Lecture, 2 Practicals
In these lectures and practicals the important classes (such as Vector, HashTable, Array and Observable) and their uses will be discussed. The Java package concept will also be introduced.
The Java Abstract Window Toolkit (AWT) - 2 Lectures, 1 Practical
The major elements of the AWT will be introduced as well as the motivation for the AWT.
Object oriented analysis and design - 1 Lecture
This lecture will introduce the object oriented analysis and design process and will briefly consider existing methodologies and the approach taken in this module.
The Unified Modeling Language (UML) - 1 Lecture
The UML is an attempt by Booch, Rumbaugh and Jacobson to produce a common language for describing OO designs. This lecture will provide an overview of the UML.
Use Case Modelling - 1 Lecture, 1 Practical
This lecture will describe Use Case Modelling; both methodological steps and the use of the UML notation.
The Object Modeling Technique (OMT) - 6 Lectures, 4 Practicals
In these lectures the OMT approach to object oriented analysis and design will be considered. The analysis phase will be discussed in detail. The UML will be used to provide the modelling notation.
