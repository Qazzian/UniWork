Baud Rate v Data Rate

  • Baud Rate
    • The number of signalling events per second.
    • The number of times per second that a data transmission channel changes state
  • Data Rate
    • The number of data bits transmitted per second
  • IF &only IF
    • each signalling event gives 1 BIT of information then Baud Rate == Data Rate
