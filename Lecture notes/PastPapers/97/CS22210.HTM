   1. (a)  Explain what we mean, in the context of operating  systems, 
           by a process.                               [5 marks]
      (b)  Explain  what  we  mean by  scheduling,  and  identify the 
           objectives of a good scheduler.                         [10]
      (c)  Discuss  the  advantages and disadvantages  that  might be 
           associated with a scheduling strategy of "shortest process 
           first".                                           [10]
 
   2. (a)  Describe what we mean by paging.                         [5]
      (b)  Outline how paging is implemented.                      [10]
      (c)  "As  main memories become very large, and processor speeds 
           increase  faster  than  discs,  paging  will  become less 
           important."  Discuss.                            [10]
 
   3. (a)  Explain  the  need  for  inter-process  communication when 
           processes  are co-operating, and describe briefly how this 
           can  be  achieved  using (i)  semaphores  or  (ii) message 
           passing.                                                  [8]
      (b)  Consider  a computer system controlling two traffic lights 
           which protect some road repairs.  There are sensors on each 
           light  to detect oncoming cars, so as to switch the  lights 
           as quickly as possible.  
           Describe  briefly the main application processes  that  you 
           might  expect  to see in such a system, discuss  how  these 
           processes  would need to communicate with each  other,  and 
           outline   how  this  might  be  achieved  using   messages.
                                                                  [17]
 
   4. (a)  Explain  what we mean by a command language, and a  command 
           language interpreter.                             [8]
      (b)  It  has been suggested that Ada might make a  good  command 
           language.   Discuss  the advantages  and  disadvantages  of 
           using Ada for this purpose.                             [17]
 
 
   5. (a)  Describe the basic facility in the Ada programming language 
           that  supports multi-programming and  illustrate,  briefly, 
           how it is used.                                           [6]
      (b)  Outline  the  process  co-operation  facilities  that   are 
           available in the Ada programming language.             [10]
      (c)  Suppose  that, within an Ada program, an array of  a  given 
           size  is  to be used as a stack.  Tasks may  push  elements 
           onto  the stack and pop elements from the stack by  calling 
           entries  in  a stack package.  Sketch the Ada code  of  the 
           stack task.                                               [9]
 
   6. (a)  What  is  meant by a  loosely-coupled  distributed  system?  
                                                                   [6]
      (b)  Describe  the  role  played  by  standards  in  distributed 
           computer systems.  Give an example.                      [9]
      (c)  NFS  is a standard for stateless distributed file  systems.  
           Discuss  the advantages and disadvantages of  statelessness 
           in the context of distributed systems.                 [10]
