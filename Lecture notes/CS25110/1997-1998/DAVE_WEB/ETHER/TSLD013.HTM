If 2.5 Km network in 5 segments then:

  • 3 coax (with stations) 21.56 bits
  • 2 links (no stations) 25.65 bits
  • 4 repeater delays 7.5 bits
  • transceiver delays 6.0 bits
  • transceiver cable delays 2.57 bits
  • etc.

Total about 499 bits worst case. Specified as 512 bits to give safety margin.
