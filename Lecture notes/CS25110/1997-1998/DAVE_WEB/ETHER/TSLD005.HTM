Notes:

notice 802.3 etc. covers part datalink, part physical

802.2 can thus be used for several styles of network

network and transport layer functions implemented by say TCP/IP

ISO model based on switched connection oriented services.

Broadcast medium doesnt have a single physical connection representing a link.

For LAN type networks some of the functions normally associated with the data link layer -eg channel allocation depend on the physical protocol used. therefor the media access mechanism is defined as part of the physical standard to allow a single uniform LLC layer.

additional notes. level 2 normal functions:

- data encapsulation (framing, addressing, error detection)

- link management (channel allocation (collision avoidance), contention resolution (collision handling))

level: - data encoding (pre amble, bit encoding)

-channel access (bit transmission, carrier sense)
