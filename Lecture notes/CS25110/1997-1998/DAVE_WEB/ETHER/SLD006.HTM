In general 4 types of service primitives between ANY two layers:

Request - requect a service

Indication significant event to higher layer possibly detection of a requect.

Response- may occur as a reply to indication

Confirm - results of last request

only unacknowledged connectionless provided by CSMA/CD so only request and indication services provided.

no response is provided by reciever (good for broadcast !)

no confirmation of arrival of data is possible

LLC has several classes of service, but onle type 1 is supported by CSMA/CD- type 1 only provides two primitives to the network layer.

For other physical level protocols more classes of service might be possible - then the LLC can provide more services to the network layer. eg confirmations for connection oriented services.
