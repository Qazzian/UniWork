Operation: Carrier Sense Multiple Access with Collision Detection. Distributed Control

  • Contend for use.
  • Acquire access and send a data packet.
  • No priority.

Simple algorithm:

1. Check if active, if yes return to 1.

2. Transmit and check for collisions.

3. Collision ? if so transmit JAM and then delay and go back to 1.

4. Transmission completed.

How could a collision occur ? ...How long is a bit ?

voltage pulse travels at 0.77 times light speed c=3.108m/s

1 bit time = 100ns (10Mbits) so size of a bit on the wire is 23m (.77*c)/10Mbps

consider time taken for signal to propagate down the wires. 2.5Km/23m...

if a station start transmitting it takes a non zero time to reach other stations.

JAM is to enable the other transmitters to realise there has been a collision- and stop transmitting.
