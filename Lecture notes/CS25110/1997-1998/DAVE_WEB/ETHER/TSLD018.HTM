Collision Backoff and Retransmission

  • Transmitter retries for a maximum of 16 attempts to send a frame.
  • The transmitter waits an integer multiple slot times determined by the following algorithm prior to each retry.
  • Rescheduling of each transmission uses “truncated binary exponential backoff”.
  • This ensures a random but increasing delay if many collisions occur.

after 2 collisions 0-4 slots
