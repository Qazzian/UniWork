Pointers to Structures and -> operator

struct mystruct new_struct ;

struct mystruct * struct_ptr;

struct_ptr = & new_struct;

then we can either write

new_struct.c = 23;

or

(*struct_ptr).c = 23;

or

struct_ptr -> c = 23;

and all are the same !
