More about Pointers….

int * p;

P can be used to point to an integer. This allocates

space for the pointer, but NOT for what it might point at.

main()

{ char *p;

*p = 'A';

}

WRONG .. P points

to nowhere...

main()

{ char *p, ch;

p = &ch;

*p = 'A';

}
