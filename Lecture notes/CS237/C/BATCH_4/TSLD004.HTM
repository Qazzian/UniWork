Cast Operator

Forced type conversions, “coercion”

(type name) expression

e.g. float x;

j=8 ; k=3;

x = j / k; /* then x is equal to 2.0 */

x= (float) j / (float) k ; /* then X is 2.66666 */

Note: cast operator has second highest precedence
