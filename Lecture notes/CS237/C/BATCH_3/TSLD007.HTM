BIT Operators [ not logical]

& is AND | is OR

^ is XOR ~ is INVERT

<< is SHIFT LEFT >> is SHIFT RIGHT

0x401 & 0xF00 value is 0x400

~0x4272 has value 0xBD8D

0x81 << 2 has value 0x204

>> SHIFT RIGHT if unsigned the top bits

filled with zeros, if signed the top bit is

duplicated.
