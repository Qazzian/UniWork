Multiple Entry Points Switch Statement

switch ( expression ) statement

has case labels

e.g.

switch ( I ) {

case 17: X = 5; break;

case 19: X=4;

case 7: X = X * 2 ; break ;

case 20: X = 0 ;

default: X = 21;

}
