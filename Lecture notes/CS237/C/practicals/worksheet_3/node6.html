Character Arrays in 'C'

As we are only just covering arrays in lectures, this section provides some simple examples of how you can declare a character array and read a string into it. Note: you MUST make sure that the array is sufficiently large to hold any string that might be provided as input. 

      char name[120];

      scanf("%s", &name[0]);

      printf("\nCompetitors name is %s\n",&name[0]);

You also need to be able to copy the contents of one character array into another to complete part two of the problem listed above. You do this using a function provided as part of the standard C libraries. You also need to include a header file to bring in definitions which are required. An example piece of code is given below. 

#include <string.h>

main ()
{ char firststring[80],secondstring[80];

    printf("Please give text :");
    scanf("%s", &firststring[0]);

    strcpy( &secondstring[0], &firststring[0] );

    printf("FirstString is %s\nSecondString is %s",
                   &secondstring[0], &firststring[0] );

}

ALSO NOTE: remember what has been said in lectures about needing the ampersand character in front of variable names provided as parameters to scanf.
