The Problem - Part Two

Your solution to this section should use an array of structures. The structures should be exactly those defined in part one, except that you should add one extra integer field which you use to hold the `competitor number'.

This problem is very similar to that of part two last week, but this time you are given a fixed number of competitors in advance, rather than reading the data as input from the user.

The competition has precisely eight competitors.

Your program should loop through the eight competitors, requesting the data for each competitor, storing each competitor's details in the array and printing out their total time etc in the format specified in part one.

When the data for all the competitors has been provided, your program should then print out the names and times of all the competitors in a table form.

Don't forget the earlier remark about designing first and don't forget to use meaningful variable names and comments.
