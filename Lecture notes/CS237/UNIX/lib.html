CS23710: Unix System Programming - Course Materials

Library Resources

It is not necessary to purchase a book for this part of CS23710. However, there are several books in the library which may be of interest to some students.

K. Haviland and B. Salama. / UNIX System Programming. / International Computer Science Series, 1994.
gives a comprehensive overview of single machine UNIX system programming, while

C. Brown. / UNIX Distributed Programming. / Prentice Hall, 1994.

goes way beyond the scope of CS23710, but some students may find its coverage of distributed programming interesting.

IEEE Software, Volume 16, Number 1, special issue about Linux.

It is also worth doing a library search using Voyager. Try electronic information services available at UWA.
