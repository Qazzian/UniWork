Bit Fields

A means to use one location and split it’s BITs

into separate accessible items.

struct mys{

unsigned p1:3,p2:4,p3:1;

int x;

} z;

/* Z is a structure of type struct mys. It has

four members, p1,p2,p3 and x. p1, p2, p3 are

bit fields and x is an integer.

CANNOT ask for address of bit fields.

i.e. &Z.p2 is ILLEGAL */
